# Easy Let's Encrypt SSL

A simple script to automatically install, issue, and renew free SSL certificates using Let's Encrypt.

## Features

* One-command SSL setup
* Automatic certificate renewal
* Supports Nginx and Apache
* Free and open source
* Beginner-friendly

## Usage

```bash
chmod +x ssl.sh
sudo ./ssl.sh
```

Follow the prompts to enter your domain and email address.

## Requirements

* Linux server
* Public domain name
* Port 80 and 443 open
